Token

O token foi carregado do navegador. Você pode copiá-lo e validar no jwt.io.

Instruções de validação manual
  1. Copie o token acima.
  2. Acesse jwt.io.
  3. Cole o token no campo Encoded (lado esquerdo).
  4. Role até JWT Signature Verification e digite: my-secret-key.
  5. Verifique se aparece Valid JWT e Signature Verified.
